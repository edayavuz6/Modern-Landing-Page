#  Sophia | Modern Philosophy Gallery

Sophia is a minimalist landing page project that blends ancient philosophical wisdom with modern web aesthetics. This project is a key piece of my portfolio, showcasing my skills in **UI/UX design** and **Frontend Development**.

##  Features
- **Modern & Sophisticated Design:** A color palette and typography chosen to reflect philosophical depth.
- **Fully Responsive:** Optimized for all devices (Mobile, Tablet, and Desktop).
- **Interactive Elements:** Smooth CSS animations and hover effects for philosopher cards.

##  Built With
- **HTML5:** Semantic structure and accessible content.
- **CSS3:** Advanced styling using Flexbox, Grid, and custom animations.
- **Figma:** Visual design and prototyping phase.
